The slip

It filled a gap itself

On a real call, the agent spoke a CPT code that was never in the dossier. Its tools were not registered, so it made one up.

The fix

Four webhook tools

Same day, we registered get_benchmark and three more. Now every number comes from a tool call, or the agent does not say it.

The audit

Checked after every call

A computed honesty audit reconciles each claim against the dossier once the call ends. Anything unsupported gets flagged.
