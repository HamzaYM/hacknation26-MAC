Implementationhonest about the hard parts

we made sure it never bluffs

It tried once. On our very first call, it made up a billing code out of thin air. So we shut that down for good.

THE SCARE

It invented a code.

A tool-less agent will make things up. We watched ours do it, live on a phone line.

THE FIX

Every number goes through a tool.

Now these are the only things it can say out loud:

get_case_briefget_benchmarkreport_lever_resultend_call_summary
THE PROOF

Real hospital prices.

We pulled Mass General's machine-readable price file. Their own published cash price, $2,633.25, is the number the agent negotiates against.

After the call, an eval re-checks every number it spoke. And it always admits it's an AI. never_deny_ai: true
