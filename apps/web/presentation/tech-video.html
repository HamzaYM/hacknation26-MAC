Guardrailsthe model is on a short leash

the code holds the line

The LLM does the talking. It does not get to decide anything that touches money or truth. Six hard rules run in code, around it.

numbers

Only through tools

get_benchmark and the dossier are the only prices it can speak. Any other number, it cannot say.

the ceiling

An impossible offer

An offer above what the patient can pay is rejected before the agent is allowed to speak it.

payment plans

Math, server-side

Monthly times months times interest is multiplied out in code. A plan that clears the ceiling only with interest is refused.

injection

Refused by policy

The rep on the phone is not its operator. Attempts to rewrite its rules get one polite no, then an exit.

disclosure

Never denies the AI

Asked if it is a bot, it says yes, every time. never_deny_ai: true, no exceptions.

after the call

Every call audited

When the call ends, a deterministic check re-reads every number it spoke. The result lands in the War Room.

new market? one file

The engine reads one YAML per market and nothing else. Medical bills today, moving quotes next, the same code underneath.

config/verticals/medical_bills.yaml red_flags: duplicate · upcode · unbundle
ladder: open → assistance → dispute → settle
personas: stonewaller · policy_citer · collections
citable_sources_only: [dossier, get_benchmark]
never_deny_ai: true

9 scenarios, keys computed by code

Every answer key is generated from real chargemaster rows and CMS rates, run through the real engine. Nothing typed by hand.
