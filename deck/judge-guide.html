A judge's guide to Haggl

What to watch for as a judge

the substance
  • It discloses being an AI whenever it is asked, and never denies it. The post-call audit fails any call where it does.
  • Every number it says is tool-gated. If a figure did not come from a tool or the case file, the agent does not say it out loud.
  • The coverage panel flips live. Required questions turn green in the War Room only once the agent has actually covered them.
  • The case file holds real phone-call audio, not a simulated stack. You can play the recordings back.
  • The /technical-architecture page explains the deterministic core: the language model is the mouth, the code is the brain.
