# Voice guide — start here

Three distinct registers, each with its own file. Don't merge them — the split is deliberate, not sprawl. This file is just the index.

| Register | File | Governs |
|---|---|---|
| **Outreach** | [voice-samples.md](voice-samples.md) | DMs, emails, any 1:1 message. Ground truth = his own real sent messages. |
| **Articles** | [article-voice-reference.md](article-voice-reference.md) | Field notes, the effort-sweep write-up, methodology posts. Third-party reference style (ClaudeDevs) + his adjustment: much more "I" than "we". |
| **Page copy** | [voice-profile-addendum.md](voice-profile-addendum.md) | Site pages — lab, builder, everything under hamz-ai-website. Derived directly from his own rewrites in feedback.pdf, 2026-07-13. |

Standing rules that apply across all three, regardless of register: no em-dashes in his own copy, no comparison/hype framing ("it's not just X, it's Y" is banned; plain two-sentence contrast is legal and is his actual voice), category term is always "AI implementation studio," never "consultancy."

If a task doesn't obviously fall into one of the three — ask, don't guess which register applies.
